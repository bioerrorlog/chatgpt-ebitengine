Call GPT in [Ebitengine](https://ebitengine.org/).

## Run on your local machine

```sh
# Set OpenAI API key
export OPENAI_API_KEY="your openai api key"

go run github.com/bioerrorlog/chatpgt-ebitengine@latest
```
